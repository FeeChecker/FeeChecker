====================

Проект служит для сбора информации об кошельках в разных сетях
ARB, ARB-nova, ETH, Fantom, Polygon, OPT, BSC, StarkNet

====================

Составляется таблица вида
Кошелек|Кол-во транз.|Первая транз.|Последняя транз.|Сред. время между транз.|Комиссия в токенах|Комиссия в долларах

Комиссия в токенах на всех кошельках
Комиссия в долларах на всех кошельках

====================

Вы можете использовать стандартные API ключи для сканеров, НО есть возможность их изменить

.models/Chain.py - файл можно открыть любым редактором текста

6-11 API ключи соответственно сетям

.core/scans/config.py

10 - api ключ для ARB-nova

api ключи можно получить в профилях сканеров
